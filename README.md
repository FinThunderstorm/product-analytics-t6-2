# Product Analytics T6.2

## Preview

Built notebook is available at <https://product-analytics-t6-2.alanen.dev>

## Data

Predictions are based on data from `Telepass: From Tolling to Mobility Platform` / `Harvard Business School Case 622-011`. Place the excel data file into `input`-folder with name `telepass.xlsx`.

## Deploying

Due to restrictions given in data file, there is no automatic building. To build for deployment, run `build.sh`. This will create needed `index.html` file into `output`-folder.
